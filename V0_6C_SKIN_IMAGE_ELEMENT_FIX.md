# 0.6c Skin Image Element Fix

- Dodano prawdziwy `<img id="skinTextureImage">` wewnątrz klikera.
- PNG nie jest już ustawiane jako CSS background.
- Gradienty i klasy skinów nie mogą nadpisać obrazu.
- Efekty oraz napis NOOB pozostają nad teksturą.
- Konsola pokaże dokładny URL, jeśli obraz się nie wczyta.
- Paczka nie zawiera żadnych tekstur.
