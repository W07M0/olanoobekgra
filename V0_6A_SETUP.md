# Ultimate Noob Clicker v0.6a

Ta aktualizacja nie wymaga nowych zmian w Supabase.

## Wgrywanie z telefonu

1. Rozpakuj paczkę.
2. Podmień całą zawartość repozytorium.
3. Upewnij się, że foldery `js`, `css` i `tests` zachowały nazwy.
4. Zrób commit.
5. Poczekaj na zielone wdrożenie GitHub Pages.
6. Otwórz stronę z `?v=06a` i odśwież.

Najważniejsze zmienione pliki:
- `index.html`
- `css/styles.css`
- `js/app.js`
- `js/data.js`
- `js/systems.js`
- `js/integrations.js`
