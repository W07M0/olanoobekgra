# 0.6c Flexible Texture Size Fix

- Tekstury nie wymagają konkretnej rozdzielczości.
- Ramki i tła zachowują proporcje i używają automatycznego kadrowania.
- Skiny dopasowują się do koła klikera.
- Dodano opcjonalny tryb `texture-contain` bez kadrowania.
- Dokładne rozmiary są teraz tylko zaleceniem jakościowym.
