# Pełny panel administratora — 0.6b

Panel pozwala edytować:
- waluty, poziom, EXP, rebirthy;
- kliknięcia, combo, czas gry i statystyki;
- kasyno;
- światy i bossów;
- skiny;
- rekordy oraz oceny minigier;
- wszystkie pety i wyposażone sloty;
- pełny zapis JSON.

## Ważne
Plik SQL `supabase_full_save_admin_pets_0_6b.sql` z poprzedniej paczki musi być uruchomiony.
Gracz musi przynajmniej raz wejść do gry po aktualizacji pełnego zapisu.

## Edytor JSON
1. Pobierz profil.
2. Edytuj JSON.
3. Kliknij `Sprawdź JSON`.
4. Kliknij `Wczytaj JSON do formularza`, aby zobaczyć najważniejsze wartości.
5. Kliknij `Zapisz zmiany profilu`.

Przed większymi zmianami skopiuj JSON jako kopię zapasową.
